# Random Password generation:
```text
* Write a program that provides a random password generation operation. Requirements for this program:
* Create an abstract class that has only one generate method.
* Create 3 classes, each created to generate a random word of length 20, one of these classes generates a random word using only a combination of letters, one using a combination of numbers, and one using a combination of numbers and letters. You are free to name the classes.
```
